# AL METRONOME

AL Metronome is a desktop application designed for musicians seeking precision and flexibility in their practice routine. Built with modern web technologies (Electron, React, and Tone.js), it delivers a high-fidelity audio engine free from drift.

Beyond standard timekeeping, it features a Step Sequencer for custom beat accentuation and a specialized Speed Trainer mode—engineered to help guitarists and instrumentalists build speed and endurance progressively.

O AL Metronome é uma aplicação desktop desenvolvida para músicos que buscam precisão e flexibilidade em sua rotina de estudos. Construído com tecnologias web modernas (Electron, React e Tone.js), ele entrega um motor de áudio de alta fidelidade e livre de oscilações (drift).

Além da marcação de tempo padrão, o software conta com um Step Sequencer para customização de acentos rítmicos e um modo de Treino de Velocidade (Speed Trainer) — projetado especificamente para auxiliar guitarristas e instrumentistas no desenvolvimento progressivo de velocidade e resistência.
